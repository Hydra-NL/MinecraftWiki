About

Het project is bedoeld om een soort wikipedia voor het spel Minecraftâ„¢ neer te zetten. De gebruiker kan deze wiki aanvullen met de volgende entiteiten: Tools, Blocks en Mobs.
Tools kunnen wapens zijn die gebruikt kunnen worden om Mobs (vaak vijandelijke monsters) te verslaan of gereedschappen om bepaalde blokken te breken.
Blocks zijn letterlijk de bouwstenen van het spel en kunnen verschillende "levels" hebben, deste hoger het level deste sterker de tool moet zijn om het blok te kunnen breken.
Mobs zijn de dieren of monsters van het spel, hiervoor zijn wapens nodig om door de bepantsering heen te kunnen komen. Ook hiervoor geldt: deste hoger het pantser-level deste sterker de tool moet zijn.

Entiteiten

Zoals in het datamodel staat aangegeven kunnen enkele entiteiten relaties met elkaar hebben. Zo heeft de Tool eigenschappen zoals "toolLevel" en "toolType", afhankelijk van het level en van het type tool, kan deze bepaalde blokken juist wel of niet breken. Tooltypes zijn bepaald aan de hand van een enum die de volgende types bevat: Pickaxe, Shovel, Axe, Sword.
Ook als een tool een wapen is moet deze een bepaald niveau van 'Attack' hebben om schade aan te kunnen richten op Mobs.

Datamodel

Voorbeelden:
Tool.attack = 3; Mob.armor = 1; Kan 2 'health' schade aanrichten.
Tool.attack = 1; Mob.armor = 3; Kan geen 'schade aanrichten.
Tool.typeTool = "Pickaxe"; Block.tool = "Shovel"; Kan het block niet breken.
Tool.typeTool = "Shovel", Tool.typeLevel = 3; Block.tool = "Shovel", Block.hardness = 5; Kan het block niet breken.
Tool.typeTool = "Axe", Tool.typeLevel = 2; Block.tool = "Axe", Block.hardness = 1; Kan het block breken.
